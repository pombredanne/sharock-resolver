# Sharock Resolver

Crystal dependency resolver.

## Requirements

- Crystal v0.9.1
- Shards v0.5.3 ~

## Build

```
$ make
```

## Usage

```
$ bin/resolver --github=OWNER/REPO --output-file=JSON_FILE_NAME --quiet
```
